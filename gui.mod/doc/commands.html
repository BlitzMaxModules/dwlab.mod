Type LTButton Extends LTLabel
DescriptionClass for button gadgets.
InformationIf button visualizer's image have more than 1 frames, they will be used following way:
  • 2 frames: first frame will be used for normal button, second - for pressed button.
  • 4 or more frames: same as above, but 3rd frame will be used for normal button with focus. and 4th - for pressed button with focus
Fields Summary
Focus , State
Field Focus:Int
DescriptionFocus of button.
InformationFocus is changed to True when mouse cursor is over the button.
Field State:Int
DescriptionState of button.
InformationState is changed to True when users press left mouse button on the button.

Type LTCheckBox Extends LTButton
DescriptionClass for button gadgets.
InformationCheckbox is the button which state changes anoter way: when user clicks checkbox first time, its state will be set to True. When user clicks checkbox second time, state returns to False and so on.

DescriptionValue by which position will change if user roll mouse wheel button on slider.

DescriptionPosition of the moving part on the slider field ( 0.0 - 1.0 ).

DescriptionType of the slider - Vertical or Horizontal.
Method Init()
DescriptionSlider initialization method.
InformationYou can set different slider properties using object parameters in editor:
  • "type" - sets slider type ( "vertical" or "horizontal" )
  • "selection" - sets slider behavior ( "moving" or "filling" )
  • "percent" - sets showing slider position in percents You can also attach sprite (which should be inside same window) as slider moving part to the slider by naming silder and set "slider_name" parameter of the sprite to slider name. And you can attach a list box (which should be inside same window) to scroll its contents with the slider by naming list box and set "list_box_name" parameter of the slider to list box name.

    DescriptionButton pressing event method.

    See also: OnButtonUnpress, OnButtonDown, OnButtonUp, OnMouseOver, OnMouseOut.

    Method OnButtonUnpress( Gadget:LTGadget, ButtonAction:LTButtonAction )
    DescriptionButton unpressing event method.
    InformationCalled when button just being unpressed on window's gadget. Some standard commands are available (can be set in editor in "action" parameter):
    • "save" - executes window's Save() method. Intended for saving data changed by window.
    • "close" - closes the window.
    • "end" - forces exit from current project.
    • "window" - opens a window with given name
    • "window_class" - opens a window of given class
    • "save_and_close" - performs "save" and "close" actions
    • "save_and_end" - performs "save" and "end" actions

    See also: OnButtonPress, OnButtonDown, OnButtonUp, OnMouseOver, OnMouseOut.

    Method OnButtonUp( Gadget:LTGadget, ButtonAction:LTButtonAction )
    DescriptionButton up event method.
    InformationCalled when button is currently released and cursor is over window's gadget.

    See also: OnButtonPress, OnButtonUnpress, OnButtonUp, OnMouseOver, OnMouseOut.

    Method OnMouseOut( Gadget:LTGadget )

    Method OnMouseOver( Gadget:LTGadget )
