Functions

Function L_ClearSoundMaps()
DescriptionCleaning function which unregisters sounds which stopped playing.
InformationThis function should be called periodically during the program execution (probably in project's Logic() method).

Function L_PlayMusic:TChannel( Sound:TSound, Temporary:Int = True, Volume:Double = 1.0, Rate:Double = 1.0, Pan:Double = 0.0, Depth:Double = 0.0 )
ReturnsChannel allocated for playing music.
DescriptionMusic playing function.
InformationUse it instead of standard sound playing functions to make profile music volume affect playing music. Function also regisers the channel of a music.

Function L_PlaySound:TChannel( Sound:TSound, Temporary:Int = True, Volume:Double = 1.0, Rate:Double = 1.0, Pan:Double = 0.0, Depth:Double = 0.0 )
ReturnsChannel allocated for playing sound.
DescriptionSound playing function.
InformationUse it instead of standard to make profile sound volume affect playing sound. Function also regisers the channel of a sound. Temporary flag defines if channel of sound should be unregistered after stopping or pausing. If you set Temporary flag to false, you should unregister sound after use manually by calling L_UnregisterSound function.

Function L_RefreshSounds()
DescriptionSound refreshing function.
InformationThis function sets the volume to all sounds. Call it when sound and music settings of current profile are changed.

Function L_SetMusicVolume( Channel:TChannel, Volume:Double )
DescriptionMusic channel volume setting function.
InformationThis function also registers the channel for setting music master volume.

Function L_SetSoundVolume( Channel:TChannel, Volume:Double )
DescriptionSound channel volume setting function.
InformationThis function also registers the channel for setting sound master volume.

Function L_UnregisterChannel( Channel:TChannel )
DescriptionCleaning function which unregisters given channel.
